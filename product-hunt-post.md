## FIRST COMMENT (Maker Comment)

Hey Product Hunt,

I'm the maker of Developer OS — a Notion template bundle I built because I kept rebuilding the same systems from scratch every time I started a new job search or side project.

Here's the honest backstory: I had a LeetCode tracker in Google Sheets, a job application list in another spreadsheet, a Notion page for my side project I kept forgetting to update, and book notes scattered across three different apps. It worked, barely. But every few months I'd lose the thread and start over.

So I spent a few weekends building a system that actually fits how developers work — not how productivity gurus think developers should work. No morning routines. No life goals tracker. Just the three things that actually matter when you're a developer trying to grow:

1. **Tech Interview Tracker** — LeetCode kanban by difficulty, a company application pipeline, and problem notes with spaced review scheduling. This one is free.

2. **Side Project Dashboard** — Sprint kanban, pre-deploy checklist, feature roadmap, and bug tracker. Built for solo developers who want to ship consistently without a PM.

3. **Developer Learning Log** — Book notes DB, a TIL log, and a weekly retrospective template. Fifteen minutes a week to actually retain what you're learning.

I use all three of these myself, which means I'll keep improving them.

Launch pricing is $39 for the full bundle (normally $49), good for two weeks.

Happy to answer any questions about how I set this up or how I'm using it day-to-day. What would make a tool like this actually useful for your workflow?
