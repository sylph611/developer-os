## Templates in Detail / 템플릿 상세

### 1. Tech Interview Tracker (무료 포함)

기술 면접 준비를 체계적으로 관리하는 트래커입니다.

- **LeetCode Kanban**: Easy / Medium / Hard 난이도별 문제 진도 관리. "Needs Review" 컬럼으로 스페이스드 리피티션 실전 적용.
- **Company Pipeline**: 지원 → 서류 → 코딩테스트 → 기술면접 → 최종면접 → 오퍼 파이프라인
- **Problem Notes DB**: 풀이 방법, 시간 복잡도, 패턴 태그, 재풀이 일정 기록
- **Interview Prep Checklist**: 회사별 면접 전 체크리스트

Tech interview prep tracker to keep your job search organized.

- LeetCode progress kanban by difficulty, with a "Needs Review" column for spaced repetition
- Company application pipeline (Applied → Screening → Technical → Offer)
- Problem notes with pattern tagging, time complexity, and review schedule
- Per-company interview prep checklist

---

### 2. Side Project Dashboard (유료)

사이드 프로젝트를 제품처럼 관리하는 대시보드입니다.

- **Sprint Kanban**: To-Do → In Progress → Done → Backlog 스프린트 보드
- **Deployment Checklist**: 프로덕션 배포 전 체크리스트 (보안, 퍼포먼스, SEO, 분석 도구)
- **Feature Roadmap**: 아이디어부터 출시까지 임팩트/난이도 스코어링 로드맵
- **Bug Tracker**: 심각도별 버그 트래킹 DB

Manage your side projects like a product, not a hobby.

- Sprint kanban with backlog management
- Pre-deploy checklist (security, performance, SEO, error tracking)
- Feature roadmap with impact vs effort scoring
- Bug tracker with severity labels

---

### 3. Developer Learning Log (유료)

꾸준한 성장을 기록하는 학습 로그입니다.

- **Book Notes DB**: 기술서적 독서 노트 (챕터별, 핵심 인사이트, 액션 아이템)
- **Weekly Retrospective**: 주간 회고 템플릿 (잘한 것 / 부족한 것 / 다음 주 목표 / 자랑스러운 것)
- **TIL Log**: Today I Learned 일일 기록 — 카테고리 태그로 검색 가능
- **Resource Library**: 아티클, 유튜브, 강의 즐겨찾기 DB

Track your growth consistently with structured reflection.

- Book notes DB with chapter-by-chapter breakdowns and action items
- Weekly retrospective template (wins / struggles / next week / proud of)
- TIL (Today I Learned) daily log, searchable by category
- Resource library for articles, videos, and courses
